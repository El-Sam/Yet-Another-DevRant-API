Yet Another DevRant API (YADevRA)
=====================================

Unofficial REST API of the cool dev community called [DevRant](https://devrant.com). 

Stack used 
----------
- Java
- Spring REST with HATEOAS
- Gradle
- JUnit
- JSoup
